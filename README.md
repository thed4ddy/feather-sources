Unofficial AltStore/Feather sources for sideloading apps.

[Plezy](https://github.com/edde746/plezy) is a modern cross-platform Plex client built with Flutter.

### How to use

1. Install [Feather](https://github.com/khcrysalis/Feather) on your iOS device
2. Tap the **Add to Feather** button above (on your iOS device)
3. Plezy will appear as a source — install and enjoy!

### Auto-updates

This source is automatically updated via GitHub Actions whenever a new Plezy release is published. You'll always get the latest version in Feather.
